Fixedpoints_demo.m:
1) Open MATLAB
2) Plug in ethernet to Hebi
3) Once connected to Hebi, run script

ROS_demo.m:
1) Open MATLAB
2) Plug in ethernet to Hebi
3) Type "rosinit" into command window of MATLAB and hit enter
4) Ensure camera code is running (following instructions for that)
5) Run MATLAB script once Hebi is connected

workspace_demo.m:
1) Open MATLAB
2) Plug in ethernet to Hebi
3) Plug in xbox controller
4) Run MATLAB script once Hebi is connected
